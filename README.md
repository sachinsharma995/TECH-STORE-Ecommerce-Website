# Tech Store E-Commerce Website

A modern **E-Commerce Shopping Cart Website** built using **React.js, Vite, and Tailwind CSS**.
This project allows users to browse products, add items to the cart, and place orders with a smooth and responsive user experience.

---

## Features

* Browse products (Laptop, Phone, Camera, Tablet)
* Search products by name
* Filter products by category
* View product details
* Add items to cart
* Increase / decrease quantity
* Remove items from cart
* Dynamic cart total calculation
* Checkout page with shipping form
* Order confirmation page
* Toast notifications for user actions (Add to cart, Remove, Order placed)
* Fully responsive design

---

## Technologies Used

* React.js (Vite)
* Tailwind CSS
* React Router DOM
* Context API (State Management)
* React Toastify (Notifications)
* JavaScript (ES6)

---

## How It Works

* Uses **Context API** to manage cart state globally
* Uses **React Router** for page navigation
* Filters products dynamically based on category
* Calculates total price using cart data
* Displays **toast notifications** for user actions
* Handles form input for checkout using controlled components
